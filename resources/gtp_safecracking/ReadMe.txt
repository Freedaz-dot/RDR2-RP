Example usage:
local result = exports.gtp_safecracking:StartSafeCrackingMiniGame(6) - The number 6 indicates the amount of tumblers that the safe has.
local result = exports.gtp_safecracking:StartSafeCrackingMiniGame({15, 23, 82, 46, 83, 59}) - The table signifies the safe code predefined by you.

'result' is a value of true or false, depending on whether you manage to open it or not.

In the configuration file, you can translate the content of prompts.
